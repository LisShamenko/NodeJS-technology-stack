Шрифты.

Использование разных шрифтов для заголовка и основной текста является распространенной практикой. Для заголовка часто используется шрифт с засечками (антиква), а для основного текста без засечек (рубленный или гротеск). Шрифты могут сильно замедлять загрузку страницы, поэтому следует выбирать только действительно нужные начертания шрифта.

Вертикальный ритм.

Вертикальный ритм это сетка из базовых линий с одинаковыми расстояниями между строками. Большая часть текста должна размещаться на этих линиях. Так текст с разными размерами шрифта и интервалами между букв создает вертикальную гармонию. Why is Vertical Rhythm an Important Typography Practice?

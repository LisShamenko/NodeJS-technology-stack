Меню nav-menu расположено там, где оно может удовлетворить потребности как мобильной, так и настольной версии. Классы row и column позволяют создать макет для настольной версии.

Для мобильной версии секции с классом column размещаются горизонтально.

Первая колонка.

Программы экранного доступа в качестве ориентиров используют элементы [form, main, nav, aside].

Вторая колонка.

Кнопка меню должна находится внутри элемента nav, иначе будет сложно найти кнопку меню через программу экранного доступа.

Третья колонка.

Кнопки, ссылки и другие ключевые области должны быть достаточно большими, чтобы их можно было легко коснуться пальцем
